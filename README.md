# DMAlertDialog

Add it in your root build.gradle at the end of repositories:


	  allprojects {
	      repositories {
		...
		maven { url 'https://jitpack.io' }
	      }
	    }
    
    Add the dependency
    
    dependencies {
	        implementation 'com.github.pmbfish40:DMAlertDialog:1.0.0'
	  }
